int main ( )

Main activity.

Program is divided into 3 parts:

Initialization: init values for visuals(vx, vy, x, y), and functionality(kx, ky, cell[3][3], inp).

Execution: the do...while() loop, running the game until a tie or someone wins.

Ending: Do some last stuff.

Phần mềm chia ra làm 3 phần:

Khởi tạo: Khởi tạo giá trị để thể hiện lên màn hình(vx, vy, x, y), thể hiện chức năng(kx, ky, cell[3][3], inp).

Thực hiện: vòng lặp do...while(), chạy game cho đến khi có người thắng hoặc hòa.

Kết thúc: thực hiện 1 số bước cuối cùng.
